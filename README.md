# 🏛️ Komputasi Weton & Falakiah Digital (v5.8)

Aplikasi berbasis web (*Single Page Application*) interaktif yang memadukan **Kaidah Falakiah Islam Klasik** (penyelarasan Kalender Hijriah) dengan **Sistem Ilmu Titen Kejawen** (Primbon Jawa & Pancasuda). 

Antarmuka dasbor dirancang dengan tema premium khas perpaduan Hijau Keraton dan warna kertas perkamen kuno. Aplikasi ini dioptimalkan penuh menggunakan CSS media breakpoints modern agar **100% responsif saat dibuka via HP** (otomatis berubah menjadi *Bottom Navigation Bar*) maupun perangkat desktop.

## 🧿 Fitur Utama Dasbor KalJa

1. **Buku Pustaka Induk Kawruh Falakiah (Menu Utama):** Hub pusat navigasi interaktif bergaya grid minimalis untuk mengakses setiap modul perhitungan kebudayaan.
2. **Hijriah & Weton:** Modul konversi penanggalan masehi ke pasaran Weton Jawa adat sekaligus Kalender Hijriah Islam secara akurat, lengkap dengan peninjauan otomatis **35 Sifat Karakter Bawaan Lahir** manusia.
3. **Jodo Weton Ganda:** Kalkulator keselarasan hubungan jodoh otomatis langsung melalui input dua tanggal lahir terpisah (Pria & Wanita). Sistem akan menghitung neptu gawan secara *real-time* menggunakan rumus **Pancasuda Jawa ($\div 5$)**, lalu mengeluarkan persentase kecocokan, rincian hitungan, serta saran tindakan preventifnya.
4. **Neptu Jeneng Nama:** Fasilitas penghitungan jumlah bobot neptu nama yang fleksibel menggunakan alfabet Latin maupun Aksara Jawa asli (Hanacaraka), dilengkapi dengan **Papan Ketik Virtual Aksara Jawa** serta fitur konversi otomatis.
5. **Hari Baik:** Sistem kalender bulanan virtual dinamis lintas milenium (**Tahun 1000 s.d. 3000**) sesuai kalender dunia nyata untuk mengaudit matriks energi harian, keblat arah mencari rezeki, serta keselamatan.

## 📜 Maklumat Peringatan Ilmu Titen
Perhitungan hisab numerik dalam aplikasi ini disadur dari manuskrip kuno klasik dan **bukan untuk diyakini sebagai takdir mutlak atau ketetapan gaib**. Hasil analisis ini hanya berfungsi sebagai sarana **"Titen"** (metode membaca pola/titik kesamaan gejala alam & perilaku sosial berdasarkan warisan pustaka spiritual leluhur). Segala kebenaran mutlak hanya milik Allah SWT.
